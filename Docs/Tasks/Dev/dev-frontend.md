# Developer A — Frontend Focus (Next.js/TS/Tailwind)

This is a suggested assignment list for a frontend-leaning developer. They can pick up a few backend tasks when unblocked.

## Weeks 1–3 (Phase 1)

- P1-SETUP-001: Initialize Next.js app structure
- P1-AUTH-001: Auth screens (signup/login/reset)
- P1-AUTH-002: Session handling + protected routes
- P1-UI-001: Global layout + navigation scaffold
- P1-DASH-001: Dashboard widgets (static)
- P1-RBAC-003: Team management UI

## Weeks 4–6 (Phase 2)

- P2-CRM-002: Contacts list + detail UI
- P2-CRM-003: Add/edit contact
- P2-ASSET-002: Upload UI + asset listing
- P2-TPL-002: Template gallery UI
- P2-CAMP-002: Campaign creation wizard (MVP)
- P2-AN-002: Campaign analytics UI

## Weeks 7–9 (Phase 3)

- P3-BUILDER-002: Drag-and-drop email editor (core)
- P3-BUILDER-003: Preview modes + send test email
- P3-INBOX-003: Inbox UI (list + thread)
- P3-WEB-002: Website admin UI
- P3-WEB-003: Page editor MVP

## Weeks 10–12 (Phase 4)

- P4-AUTO-003: Automation builder UI (MVP)
- P4-BILL-004: Plan gating UI hooks
- P4-AB-002: A/B testing UI
- P4-PUB-002: Publishing UI
- P4-AN-001: Unified analytics dashboard

## Notes

- Coordinate early with Backend on data shapes for: templates, builder JSON, automations JSON.

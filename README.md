# Nanis — Campaign & Growth Management SaaS

All-in-one Campaign, Communication, and Growth Management platform ("Mailchimp on Steroids"). Multi-tenant SaaS with email campaigns, visual builder, automations, unified inbox, website builder, analytics, and subscription billing.

## Repo structure

- **`frontend/`** — Next.js + TypeScript + Tailwind (UI, routing, auth screens)
- **`backend/`** — Supabase (Postgres, Auth, Storage, Edge Functions), API
- **`Docs/`** — Project docs, tasks, API contract, and team collaboration

## Tech stack

- **Frontend:** Next.js, TypeScript, Tailwind CSS, Framer Motion
- **Backend:** Supabase (PostgreSQL, Auth, Storage, Realtime, Edge Functions)
- **Email:** Amazon SES | **Payments:** Razorpay
